Role-scoped, not practice-scoped
Every domain is built for one type of person doing one type of work. When a Wealth Manager opens TrueMath and selects their domain, everything they need is there. They never wonder which domain has the function they want.
Self-contained toolkits
Each domain is a complete calculation environment. Activate one or many. Each session starts with a domain selected, and all variables and activities for that role are immediately available.
Starting points, not locked configs
Library domains are foundations. Modify terminology, adjust default values, extend activities, add visualizations, or build entirely new domains for workflows unique to your firm.

Change how TrueMath recognizes terms in your prompts. Use your firm's specific terminology, not generic definitions.

Adjust defaults
Set default values that reflect your typical client profile, market assumptions, or firm-wide standards.
03
Add activities
Extend any domain with new calculation activities for workflows the Library version doesn't cover.
04
Build visualizations
Design outputs for your highest-value client moments — charts, tables, scenario comparisons built to be shared.
05
Create new domains
Build entirely new domains for practice areas unique to your firm. The TrueMath Assistant handles the construction.
06
Share across your team
Customized domains can be shared across your organization, so your whole team operates from the same calculation foundation.
